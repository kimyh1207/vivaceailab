# 2장. 프론트엔드 기초: HTML · CSS · JavaScript

## 들어가며

백엔드 개발자도 프론트엔드를 알아야 합니다. AI 서비스를 만들어도 사용자가 쓰는 건 결국 화면입니다. 버튼 하나, 입력창 하나를 이해하지 못하면 서비스 전체를 설계할 수 없습니다.

이 챕터는 프론트엔드 전문가를 목표로 하지 않습니다. **웹이 어떻게 작동하는지 이해하고, 필요한 화면을 직접 만들 수 있는 수준**을 목표로 합니다.

---

> HTML은 뼈대, CSS는 옷, JavaScript는 행동입니다. 세 가지가 맞물릴 때 비로소 화면이 살아납니다.

이 챕터를 마치면 API 응답을 받아 화면에 렌더링하는 간단한 웹 페이지를 직접 만들 수 있게 됩니다. 그 경험이 이후 Spring Boot, AI 백엔드와 연결되는 다리가 됩니다.
